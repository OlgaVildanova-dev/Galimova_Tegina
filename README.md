# GitHub User Finder

**Автор:** Галимова Регина

## Описание
Приложение для поиска пользователей GitHub с возможностью добавления в избранное и сохранения в JSON.

## Как использовать API
1. Введите логин пользователя в поле поиска.
2. Нажмите «Поиск».
3. Выберите пользователя и добавьте в избранное.
4. Избранные пользователи сохраняются в `favorites.json`.

## Примеры использования
- Поиск существующего пользователя: введите `torvalds`.
- Добавление в избранное: нажмите кнопку рядом с пользователем.
